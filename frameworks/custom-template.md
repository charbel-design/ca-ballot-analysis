# Framework: [Your Framework Name]

## Purpose of this template

Copy this file, rename it, and fill it in to build your own framework. The structure is opinionated — it forces you to make your values explicit rather than hiding them in vague language.

If you skip sections, the framework won't work. The skill needs all sections filled to produce useful output.

## Overview

**One sentence**: What does this framework value above all else?

**One paragraph**: Why? What's the underlying theory of how government should work, what policies should prioritize, and what trade-offs you accept?

**Honesty check**: What values does this framework NOT optimize for? What will it get wrong? Who will hate this framework's outputs?

## Framework test

Before using this framework on a real ballot, test it by predicting the pick on 2-3 past races where you already know what your instinct would be. If the framework produces a pick that doesn't match your instinct, either:
1. Your instinct is wrong (which happens — frameworks are for cases where instinct is unreliable)
2. Your framework is wrong (needs adjustment)
3. The race is genuinely hard (framework works as intended but the answer is uncertain)

Be willing to update the framework if #2, not #1 or #3.
